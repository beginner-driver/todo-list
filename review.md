## 1. 총평

React + TypeScript + Vite로 구성하고, 실제 서버 없이 MSW(mock service worker)로 REST API를 시뮬레이션하고 localStorage를 저장소로 쓰는 구조다. 요구 기능은 전부 구현되어 있고 테스트 18개, 빌드 모두 정상 동작해 완성도가 높다. 다만 README의 배포 링크가 저장소 리네임 이전 경로를 그대로 가리키고 있어 문서 수정이 필요하고, 회고(배운 점/아쉬운 점 등) 기록은 아직 없다.

### 2.3 발견된 이슈 — 배포 링크 불일치

README.md 5번째 줄의 배포 링크가 `https://beginner-driver.github.io/todo-list/`로 되어 있는데, 커밋 `fix: 저장소 이름 변경(3_todo-list)에 맞춰 vite base 경로 수정`에서 저장소명이 `3_todo-list`로 바뀌면서 `vite.config.ts`의 `base` 값도 `/3_todo-list/`로 변경되었다. 즉 README의 링크는 실제 배포 경로(`/3_todo-list/`)와 다른, 존재하지 않는 경로를 가리키고 있다. `https://beginner-driver.github.io/3_todo-list/`로 수정 필요.
